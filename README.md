# Node-egg-mongoDb
超详细Node项目注解，egg框架行级代码注释，mongoose集成数据库操作，网站企业级安全，nginx代理服务，redis数据缓。涉及权限管理，用户登录到登出的闭环，为node学习以及深入研究作为开放初衷

1、下载项目运行 npm install
2、npm run dev 运行项目
